

--- Installing

0. conda create --name py37 python=3.7
1. conda activate py37 
2. pip uninstall dispel4py
3. git clone https://github.com/zhtisi/dispel4py.git 
4. git checkout dynamic_redis_stateful_dev
5. cd dispel4py
6. python setup.py install

Or 

1. conda create --name py37 python=3.7
2. conda activate py37
3. tar -zxvf dispel4py.tar . 
4. cd dispel4py
5. python setup.py install

---

pip install redis or brew install redis
